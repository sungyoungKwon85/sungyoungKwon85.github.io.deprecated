  **새로운 저장소 만들기**   
    1. mkdir directory  
    2. cd directory  
    3. git init  

  **로컬 저장소 복제**  
    `git clone /로컬/저장소/경로`  
  **원격 저장소 복제**  
  `git clone 사용자명@호스트:/원격/저장소/경로`  



  **인덱스에 추가**  
    `git add <file name>`  
    `git add *`  

  **HEAD에 반영**  
    `git commit -m "description"`  

  **원격 서버에 반영**  
    `git push origin master` // 또는 branch name 또는 원격 서버 주소 기재할 것  

  **브랜치 갈아타기**  
    `git checkout -b branchname`  
  **브랜치 변경**  
    `git checkout master`  
  **브랜치 삭제**  
    `git branch -d branchname`  

  **로컬을 원격저장소와 맞추기**  
    `git push`  

  **다른 브랜치를 현재 브랜치로 옮기기**  
    `git merge branchname`  

  **비교하기**  
    `git diff orgbranch desbranch`
